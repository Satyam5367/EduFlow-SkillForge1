# How to Add Screenshots

1. Run the project locally: `npm run dev`
2. Take screenshots of:
   - `home.png`        — Landing page
   - `courses.png`     — Course catalog with filters
   - `course-detail.png` — Course detail + checkout
   - `learn.png`       — Video player + AI quiz
   - `instructor.png`  — Instructor dashboard
   - `admin.png`       — Admin dashboard
   - `swagger.png`     — API docs at /api-docs

3. Save them in this folder as 1280x720px PNG files

4. They will automatically appear in the README.

**Tip:** Use a tool like LightShot, Greenshot, or browser DevTools device simulation for consistent sizing.
