## 🚀 Welcome

EduFlow – SkillForge is a modern, production-ready Learning Management System (LMS) designed to provide an end-to-end online learning experience.

From secure authentication and course management to AI-powered quizzes and online payments, the platform demonstrates a scalable full-stack architecture suitable for real-world educational platforms.
